# Инструкция для обновления бэкенда

## 1. Добавить колонку в БД

Выполнить SQL-запрос:

```sql
ALTER TABLE fundraisers ADD COLUMN IF NOT EXISTS image_urls TEXT[];
```

## 2. Перезапустить сервер

```bash
cd backend
npm start
```

## Что изменилось

1. **Максимальный размер файла**: увеличен до 5MB
2. **Загрузка нескольких фото**: до 5 файлов за раз
3. **Поле image_urls**: массив URL-адресов загруженных изображений

## Проверка

После запуска создай сбор с фотографиями - должно работать.
