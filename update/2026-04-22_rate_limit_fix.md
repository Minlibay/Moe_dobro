# Увеличение rate limit для админа

**Дата:** 22.04.2026

## Проблема

Админ получает 429 ошибку "Слишком много попыток" при частых запросах.

## Решение

В файле `backend/src/server.js`:

```javascript
// Более строгий лимит для авторизации (сейчас 50)
const authLimiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 минут
  max: 200, // увеличить с 50 до 200
  message: { error: 'Слишком много попыток входа, попробуйте через 15 минут' }
});
```
