## Технические детали

### Валидация

**СБП:**
- Номер телефона: обязательное, формат +7XXXXXXXXXX
- Банк: обязательное, выбор из списка

**Карта:**
- Номер карты: обязательное
- Владелец карты: обязательное
- Банк: необязательное

### Интеграция с backend

Метод `createFundraiser` в `FundraiserProvider` передаёт:
```dart
paymentMethod: _paymentMethod,
cardNumber: _paymentMethod == 'card' ? _cardNumberController.text : null,
cardHolderName: _paymentMethod == 'card' ? _cardHolderController.text : null,
bankName: _paymentMethod == 'card' && _bankController.text.isNotEmpty ? _bankController.text : null,
sbpPhone: _paymentMethod == 'sbp' ? _sbpPhoneController.text : null,
sbpBank: _paymentMethod == 'sbp' ? _selectedBank : null,
```

## Тестирование

Для тестирования:
1. Запустите приложение
2. Войдите в аккаунт (или создайте новый)
3. Перейдите в "Создать сбор"
4. Переключайте между "Карта" и "СБП"
5. Заполните форму и создайте сбор
6. Откройте созданный сбор
7. Нажмите "Помочь" и проверьте отображение реквизитов
