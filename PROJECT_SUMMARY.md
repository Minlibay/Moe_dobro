## 🎯 Основные функции

### Для пользователей
1. **Просмотр сборов** - список активных сборов с фильтрацией
2. **Помощь людям** - перевод от 100₽ с подтверждением скриншотом
3. **Создание сбора** - после первого пожертвования
4. **Статистика** - сколько помог, сколько получил
5. **Достижения** - награды за добрые дела

### Для организаторов сборов
1. **Создание сбора** - описание, цель, реквизиты
2. **Проверка донатов** - подтверждение/отклонение
3. **Отслеживание прогресса** - процент сбора
4. **Уведомления** - о новых пожертвованиях

## 🚀 Как запустить

### Быстрый старт (Windows)
```bash
start.bat
cd mobile
flutter run
```

### Быстрый старт (Linux/Mac)
```bash
chmod +x start.sh
./start.sh
cd mobile
flutter run
```

### Пошагово
1. **База данных**: `docker-compose up -d`
2. **Backend**: `cd backend && npm install && npm run dev`
3. **Flutter**: `cd mobile && flutter pub get && flutter run`

## 🔄 Workflow

1. **Регистрация** → Пользователь создает аккаунт
2. **Просмотр** → Видит активные сборы
3. **Пожертвование** → Переводит деньги, прикладывает скриншот
4. **Подтверждение** → Организатор проверяет и подтверждает
5. **Достижение** → Автоматически начисляется награда
6. **Создание сбора** → Теперь может создать свой сбор

## 🐛 Известные ограничения

1. **Нет SMS верификации** - упрощенная регистрация
2. **Нет платежных систем** - только ручные переводы
3. **Нет модерации** - все сборы публикуются сразу
4. **Локальный сервер** - для продакшена нужен хостинг
