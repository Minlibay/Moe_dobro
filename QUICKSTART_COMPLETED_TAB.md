## 🚀 Как запустить и протестировать

### 1. Backend уже запущен ✅
```bash
# Backend работает на http://localhost:3003
# База данных работает (Docker)
```

### 2. Запустить Flutter приложение
```bash
cd mobile
flutter run
```

### 3. Проверить функциональность
1. Откройте приложение
2. Нажмите на третью вкладку внизу "Завершенные" (иконка с галочкой)
3. Вы увидите список завершенных сборов с:
   - Отчетными документами
   - Категориями (🏠 Ипотека, 💊 Лечение, 📚 Образование)
   - Сообщениями о завершении
   - Статистикой

## 📊 Тестовые данные

В базе есть 5 завершенных сборов:
1. **Запуск** (Ипотека) - 1,000,000₽
2. **Помощь на лечение** (Лечение) - 125,000₽
3. **Помощь на лечение мамы** (Лечение) - 150,000₽
4. **Оплата обучения в университете** (Образование) - 85,000₽
5. **Первый взнос по ипотеке** (Ипотека) - 520,000₽

## 🎯 Особенности

### Приоритет отображения
- Завершенные сборы **изолированы** в отдельной вкладке
- Не мешают активным сборам на главной странице
- Всегда в самом низу по приоритету (отдельная вкладка)

### Что видно в карточке
- ✅ Отчетный документ (фото подтверждения)
- ✅ Категория с эмодзи
- ✅ Статус "Завершен" (зеленый бейдж)
- ✅ Название сбора
- ✅ Сообщение организатора
- ✅ Собранная сумма
- ✅ Количество донатов
- ✅ Дата завершения

## 🔧 API Endpoint

```bash
# Получить все завершенные сборы
curl http://localhost:3003/api/fundraisers?status=verified_completed

# С фильтром по категории
curl http://localhost:3003/api/fundraisers?status=verified_completed&category=medical
```
