# Моё добро - Приложение для взаимопомощи

P2P платформа для сбора средств на важные цели: ипотека, лечение, образование и другое.

## 🚀 Технологии

- **Frontend**: Flutter (iOS/Android)
- **Backend**: Node.js + Express
- **Database**: PostgreSQL (Docker)
- **State Management**: Provider
- **API**: REST

## 📦 Установка и запуск

### 1. Запуск базы данных

```bash
docker-compose up -d
```

### 2. Запуск backend

```bash
cd backend
npm install
cp ../.env.example .env
npm run dev
```

### 3. Запуск Flutter приложения

```bash
cd mobile
flutter pub get
flutter run
```

## 🎯 Основные функции

- ✅ Регистрация и авторизация по номеру телефона
- ✅ Просмотр активных сборов
- ✅ Создание собственного сбора (после первого доната)
- ✅ Перевод средств с подтверждением скриншотом
- ✅ Личный кабинет со статистикой
- ✅ Система достижений
- ✅ Уведомления
