# Моё добро - Мобильное приложение

Flutter приложение для P2P платформы взаимопомощи.

## Установка

1. Установите зависимости:
```bash
flutter pub get
```

2. Запустите приложение:
```bash
flutter run
```

## Структура проекта

```
lib/
├── config/          # Конфигурация API
├── models/          # Модели данных
├── providers/       # State management (Provider)
├── screens/         # Экраны приложения
│   ├── auth/        # Авторизация
│   ├── home/        # Главный экран
│   ├── fundraiser/  # Сборы
│   ├── profile/     # Профиль
│   └── achievements/# Достижения
├── services/        # API сервисы
├── widgets/         # Переиспользуемые виджеты
└── main.dart        # Точка входа
```

## Основные функции

- ✅ Онбординг
- ✅ Регистрация/Вход
- ✅ Просмотр сборов с фильтрацией
- ✅ Детальная информация о сборе
- ✅ Создание пожертвования со скриншотом
- ✅ Создание собственного сбора
- ✅ Личный кабинет со статистикой
- ✅ Система достижений
- ✅ Красивый UI/UX дизайн
