## 🎨 КОМПОНЕНТЫ

### 1. **Login.jsx**
- Форма входа с телефоном и паролем
- Проверка прав администратора
- Сохранение токена в localStorage
- Редирект на главную после входа

### 2. **Dashboard.jsx**
- 6 карточек со статистикой:
  - Пользователи (всего / верифицированы)
  - Сборы (всего / активные)
  - Донаты (всего / ожидают)
  - Всего собрано (сумма / средний донат)
  - Одобрено донатов
  - Ожидают донатов
- **Графики с Recharts:**
  - LineChart - донаты за последние 30 дней
  - PieChart - популярные категории
- Таблица статистики по категориям

### 3. **Users.jsx**
- Таблица всех пользователей
- Поиск по имени и телефону
- Отображение:
  - ID, имя, телефон
  - Сумма донатов
  - Сумма полученных средств
  - Статус верификации
- Кнопка верификации/снятия верификации

### 4. **Fundraisers.jsx**
- Таблица всех сборов
- Фильтры:
  - Поиск по названию
  - Статус (active/completed/closed)
  - Категория (mortgage/medical/education/other)
- Отображение:
  - ID, название, категория
  - Цель и собранная сумма
  - Статус
  - Иконка звезды для избранных
- Меню действий (три точки):
  - 🔒 Закрыть сбор
  - ⭐ Добавить/убрать из избранного
  - 🗑️ Удалить сбор

### 5. **Donations.jsx**
- Таблица всех донатов
- Фильтр по статусу (pending/approved/rejected)
- Отображение:
  - ID, от кого, кому
  - Название сбора
  - Сумма
  - Статус с иконками
  - Дата создания

## 🎯 ФУНКЦИОНАЛЬНОСТЬ

### Навигация
- Боковое меню (sidebar) с 4 разделами:
  - 📊 Панель (Dashboard)
  - 👥 Пользователи (Users)
  - ❤️ Сборы (Fundraisers)
  - 💰 Донаты (Donations)
- Кнопка выхода внизу

### Защита маршрутов
- ProtectedRoute компонент
- Проверка наличия admin_token
- Автоматический редирект на /login

### API интеграция
- Axios клиент с interceptors
- Автоматическое добавление токена в заголовки
- Обработка 401 ошибки (автоматический logout)
- Все методы из backend API:
  - login, getProfile, logout
  - getStats, getUsers, verifyUser
  - getFundraisers, closeFundraiser, featureFundraiser, deleteFundraiser
  - getDonations

## 🚀 ЗАПУСК

### 1. Установка зависимостей:
```bash
cd admin-web
npm install
```

### 2. Запуск dev сервера:
```bash
npm run dev
```

### 3. Открыть в браузере:
```
http://localhost:5173
```

### 4. Вход:
- Телефон: +79991234567 (или ваш админ)
- Пароль: ваш пароль
- Должен быть is_admin = TRUE в БД
