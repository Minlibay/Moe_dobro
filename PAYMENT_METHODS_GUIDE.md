# 💳 Руководство по методам оплаты

## Для пользователей

### Создание сбора

При создании сбора вы можете выбрать один из двух способов получения средств:

#### 1. 💳 Банковская карта
- Укажите номер карты (16 цифр)
- Укажите владельца карты (как написано на карте)
- Опционально: укажите название банка

**Когда использовать:**
- Если у вас нет СБП
- Если вы хотите получать переводы на конкретную карту

#### 2. 📱 СБП (Система быстрых платежей)
- Укажите номер телефона в формате +7XXXXXXXXXX
- Выберите банк из списка

**Преимущества СБП:**
- ✅ Мгновенные переводы 24/7
- ✅ Без комиссии для физических лиц
- ✅ Не нужно вводить номер карты
- ✅ Работает во всех банках РФ
- ✅ Безопасность (номер карты не раскрывается)

**Когда использовать:**
- Если у вас подключён СБП в банковском приложении
- Если вы хотите получать переводы быстрее
- Если вы хотите избежать комиссий

### Как сделать пожертвование

1. Откройте сбор
2. Нажмите кнопку "Помочь"
3. Вы увидите реквизиты для перевода:
   - Для СБП: номер телефона и банк
   - Для карты: номер карты и владелец
4. Нажмите на иконку копирования, чтобы скопировать реквизиты
5. Сделайте перевод в своём банковском приложении
6. Сделайте скриншот перевода
7. Вернитесь в приложение и прикрепите скриншот
8. Отправьте на проверку

## Для разработчиков

### API

**Создание сбора с СБП:**
```json
POST /api/fundraisers
{
  "title": "Название",
  "description": "Описание",
  "category": "medical",
  "goal_amount": 100000,
  "payment_method": "sbp",
  "sbp_phone": "+79991234567",
  "sbp_bank": "Сбербанк"
}
```

**Создание сбора с картой:**
```json
POST /api/fundraisers
{
  "title": "Название",
  "description": "Описание",
  "category": "medical",
  "goal_amount": 100000,
  "payment_method": "card",
  "card_number": "1234 5678 9012 3456",
  "card_holder_name": "IVAN IVANOV",
  "bank_name": "Сбербанк"
}
```

### Модель данных

```dart
class Fundraiser {
  final String paymentMethod; // 'sbp' или 'card'
  final String? cardNumber;
  final String? cardHolderName;
  final String? bankName;
  final String? sbpPhone;
  final String? sbpBank;
}
```

### Валидация

**Backend (fundraiserController.js):**
- СБП: требует `sbp_phone` и `sbp_bank`
- Карта: требует `card_number`
- CHECK constraint на уровне БД

**Frontend (create_fundraiser_screen.dart):**
- СБП: валидация формата телефона `+7XXXXXXXXXX`
- Карта: проверка заполнения обязательных полей

### База данных

```sql
CREATE TYPE payment_method_enum AS ENUM ('sbp', 'card');

ALTER TABLE fundraisers
  ADD COLUMN payment_method payment_method_enum NOT NULL DEFAULT 'card',
  ADD COLUMN sbp_phone VARCHAR(20),
  ADD COLUMN sbp_bank VARCHAR(100),
  ALTER COLUMN card_number DROP NOT NULL,
  ALTER COLUMN card_holder_name DROP NOT NULL;

ALTER TABLE fundraisers
  ADD CONSTRAINT check_payment_details CHECK (
    (payment_method = 'sbp' AND sbp_phone IS NOT NULL AND sbp_bank IS NOT NULL) OR
    (payment_method = 'card' AND card_number IS NOT NULL)
  );
```

## Безопасность

- Все реквизиты хранятся в зашифрованном виде
- Номера карт и телефонов не передаются третьим лицам
- Скриншоты переводов проверяются модераторами
- Пожертвования подтверждаются вручную

## FAQ

**Q: Можно ли изменить метод оплаты после создания сбора?**
A: Нет, метод оплаты нельзя изменить после создания. Создайте новый сбор с нужным методом.

**Q: Какие комиссии берёт приложение?**
A: Приложение не берёт комиссий. Все средства идут напрямую получателю.

**Q: Как быстро приходят деньги?**
A: При переводе по СБП — мгновенно. При переводе на карту — зависит от банка (обычно до 10 минут).

**Q: Что делать, если я указал неверные реквизиты?**
A: Свяжитесь с поддержкой через профиль. Мы поможем исправить ошибку.

**Q: Можно ли получать средства на карту другого человека?**
A: Да, но убедитесь, что у вас есть доступ к этой карте для получения средств.
