## 📝 API ПРИМЕРЫ

### Получить статистику:
```bash
curl -X GET http://127.0.0.1:3003/api/admin/stats \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

**Ответ:**
```json
{
  "overview": {
    "total_users": 150,
    "verified_users": 45,
    "total_fundraisers": 80,
    "active_fundraisers": 35,
    "completed_fundraisers": 40,
    "total_donations": 250,
    "approved_donations": 230,
    "pending_donations": 12,
    "total_amount": "450000.00",
    "avg_donation": "1956.52"
  },
  "daily_stats": [...],
  "top_categories": [...]
}
```

### Верифицировать пользователя:
```bash
curl -X PATCH http://127.0.0.1:3003/api/admin/users/10/verify \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"is_verified": true}'
```

### Закрыть сбор:
```bash
curl -X PATCH http://127.0.0.1:3003/api/admin/fundraisers/5/close \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

## 🎯 КАК ИСПОЛЬЗОВАТЬ

```sql
UPDATE users SET is_admin = TRUE WHERE id = YOUR_USER_ID;
```
